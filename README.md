A work in progress project. This is a Reactjs Web App that let you search for books listed on Google Books. In this project, Google Book API was used.


## Prerequisites
1. Node.js (version 6 or higher)
2. A Google Book API. (Usee this API link: https://www.googleapis.com/books/v1/volumes'.) Read more here: https://developers.google.com/books/docs/v1/getting_started
3. Install Superagent with `npm install superagent` - this let you get data you needed from the API. read more here:
